# 02 — 功能模块与优先级

> 所属：Neo Concept 结构化需求文档

---

## 优先级定义

| 级别 | 含义 |
|------|------|
| **P0** | MVP 必须 — 无此功能 App 不可用 |
| **P1** | 重要 — 上线后尽快补 |
| **P2** | 增强 — 有则更好 |

---

## 功能优先级总览

| 模块 | 功能 | 优先级 |
|------|------|--------|
| 课程管理 | 获取 manifest 索引、按需下载单课 JSON、本地缓存 | P0 |
| 课程管理 | ECDICT 数据库初始化（从 APK assets 复制 .db 文件到 SQLite 目录） | P0 |
| 课程管理 | 检查更新（Hash 对比）、增量下载 | P1 |
| 课程管理 | 下载队列管理（排队序号 + 取消下载） | P1 |
| 学习流程 | 线性模式（解锁制）+ 自由模式 | P0 |
| 学习流程 | Step 1-6 完整学习闭环 | P0 |
| 学习流程 | 每 Step 自动保存进度 | P0 |
| 学习流程 | 中断恢复（Per-Step Caching） | P0 |
| 学习流程 | 已完成 Step 回看（只读）+ 继续/回顾过渡界面 | P1 |
| 设置 | 学习模式、TTS 速度、口语阈值 | P0 |
| 内联词汇查阅 | 所有英文文本单词点击弹 tooltip | P0 |
| 学习统计 | 总学习时长、完成课时数、最近学习记录 | P1 |
| 数据管理 | 导出学习数据为 JSON/ZIP | P1 |
| 数据管理 | 从文件导入恢复数据 | P1 |
| TTS 语音 | 全文朗读、逐句朗读、可调速 0.5x-1.5x | P0 |
| ASR 识别 | 口语跟读录音 + 识别 + 编辑距离评分 | P0 |
| AI 配图 | 课文配图显示 + 本地缓存 | P2 |
| 离线能力 | 全部学习功能无需联网（课程 JSON 下载除外） | P0 |
