# 05 — 用户故事：模块 C — 练习交互详情 (Step 1-6)

> 所属：Neo Concept 结构化需求文档

---

## US-C1 — Step 1 课文预习

> **作为** 学习者
> **我希望** 预习课文时能看配图、听朗读、查单词、了解语法点
> **以便** 在上手练习前对课文有整体理解

### 验收标准

- [ ] AC-C1.1：显示 AI 配图（JSON 中 `images` 字段，本地缓存），无图时优雅降级（仅显示课文）
- [ ] AC-C1.2：课文全文显示，所有单词可点击弹出释义 tooltip
- [ ] AC-C1.3：TTS 全文朗读，播放时当前句高亮
- [ ] AC-C1.4：TTS 速度可调（0.5x / 0.75x / 1.0x / 1.25x / 1.5x）
- [ ] AC-C1.5：逐句播放：点击某句 → 朗读该句 → 该句高亮
- [ ] AC-C1.6：显示语法点卡片（本课涉及的语法点 + 简要说明 + 例句）

### 异常流程

- **TTS 模型未加载**：提示「语音引擎正在准备中...」，自动异步加载
- **图片加载失败**：显示占位图 + 「图片加载失败」，不阻塞学习
- **ECDICT 数据库未就绪**：单词仍可点击，tooltip 显示「词典加载中...」

## US-C2 — Step 2 填词练习

> **作为** 学习者
> **我希望** 在课文句子的空格中填入正确的词
> **以便** 在语境中巩固词汇和语法

### 验收标准

- [ ] AC-C2.1：显示 6-8 个含空格的课文句子（每句 1 空，从课文不同位置抽取）
- [ ] AC-C2.2：每空提供提示：首字母 + 词性（n./v./adj.）+ 中文释义
- [ ] AC-C2.3：用户输入后实时比对，正确 → 绿色 ✅，错误 → 显示正确答案（红色）+ 发音按钮
- [ ] AC-C2.4：全部完成后显示总分 + 错误词列表
- [ ] AC-C2.5：总正确率 ≥ 60% 视为通过；< 60% 允许重做

### 异常流程

- **用户输入为空就提交**：提示「请输入答案」
- **用户输入特殊字符/数字**：正常比对，不额外限制（输入什么比什么）

### 边界场景

- 大小写不敏感（"Sunday" 和 "sunday" 都算对）
- 首尾空格 trim 后再比对

## US-C3 — Step 3 词汇记忆

> **作为** 学习者
> **我希望** 通过闪卡、拼写、匹配三种方式学习新词汇
> **以便** 从认知 → 拼写 → 语义多维度掌握词汇

### 验收标准 — 闪卡子模式

- [ ] AC-C3.1：正面显示英文 + 发音按钮，反面显示中文 + 例句
- [ ] AC-C3.2：左滑 = 不认识，右滑 = 认识
- [ ] AC-C3.3：「跳过认识词」开关，开启后仅展示标记为「不认识」的词
- [ ] AC-C3.4：错词自动加入复习队列

### 验收标准 — 拼写子模式

- [ ] AC-C3.5：TTS 发音 → 用户输入拼写 → 即时反馈
- [ ] AC-C3.6：可重听发音（不限次数）
- [ ] AC-C3.7：错误显示正确拼写，红色标注差异字符

### 验收标准 — 英汉匹配子模式

- [ ] AC-C3.8：拖拽或点击配对，英文→中文
- [ ] AC-C3.9：计时完成，显示用时

### 完成条件

三个子模式按顺序进行，**全部走完后 Step 3 标记为 completed**：
- 闪卡：所有卡片滑动完毕即完成（无论认识与否）
- 拼写：全部 6-8 词拼写完毕即完成；正确率 ≥ 60% 视为通过，< 60% 可重做拼写子模式
- 匹配：全部配对完毕即完成

### 异常流程

- **TTS 发音加载失败（拼写模式）**：显示单词文本替代，不阻塞拼写
- **闪卡数据为空（Agent 遗漏）**：显示提示「暂无词汇数据」，可跳过进入拼写
- **匹配拖动过程中松手**：卡片回弹到原位，不判定对错

## US-C4 — Step 4 听力练习

> **作为** 学习者
> **我希望** 通过听 TTS 朗读的课文片段回答选择题
> **以便** 训练听力抓取信息的能力

### 验收标准

- [ ] AC-C4.1：TTS 朗读课文片段（非完整课文，是截取的关键段落）
- [ ] AC-C4.2：4-5 道选择题，每题 4 个选项
- [ ] AC-C4.3：可重复播放（不限次数）
- [ ] AC-C4.4：题型分布：主旨题 1 + 细节题 2-3 + 推理题 1
- [ ] AC-C4.5：即时判分，错题可点击「重听」重新播放对应片段

### 异常流程

- **TTS 朗读中断（切后台等）**：恢复后音频停止，用户手动重播

### 边界场景

- 播放音频时用户点击下一题 → 停止当前音频，切换题目

## US-C5 — Step 5 阅读理解

> **作为** 学习者
> **我希望** 阅读课文后回答选择题，且错题能定位到原文对应位置
> **以便** 理解错误原因，提升阅读理解能力

### 验收标准

- [ ] AC-C5.1：展示课文全文（可滚动）
- [ ] AC-C5.2：5 道选择题：细节理解 2 + 词义推断 1 + 推理判断 1 + 主旨大意 1
- [ ] AC-C5.3：选错后自动高亮原文对应句（`evidence` 字段），显示解析
- [ ] AC-C5.4：即时判分

### 异常流程

- **evidence 字段缺失**（Agent 生成遗漏）：不崩溃，仅显示正确选项，不高亮原文

## US-C6 — Step 6 口语练习

> **作为** 学习者
> **我希望** 跟读课文句子并获得发音准确度评分
> **以便** 知道自己的发音与标准的差距

### 验收标准

- [ ] AC-C6.1：3-5 句，难度递进（短句→中等→长句）
- [ ] AC-C6.2：显示句子 → 播放 Piper TTS 标准发音 → 用户录音 → Whisper 识别
- [ ] AC-C6.3：编辑距离比对，显示准确率百分比
- [ ] AC-C6.4：评分公式：`accuracy = max(0, 1 - edit_distance / max(len(target), len(recognized))) × 100%`；结果四舍五入到整数
- [ ] AC-C6.5：标红偏差词汇（用户说的 vs 原文差异）
- [ ] AC-C6.6：最低可接受准确率 60%（可在设置中调整）
- [ ] AC-C6.7：低于阈值可重录，不限次数

### 异常流程

- **ASR 模型未加载**：提示「语音识别引擎准备中...」，自动异步加载，加载完前禁用录音按钮
- **录音权限未授予**：提示「请在系统设置中授予录音权限」
- **识别结果为空（没说话/声音太小）**：提示「未检测到语音，请重试」
- **环境噪声过大**：识别结果可能不准确，这是已知限制（初版不做降噪处理）

### 边界场景

- 句子极短（< 3 个词）：编辑距离评分可能偏高，提示适当放宽标准
- 录音时长超过 30 秒：自动停止录音
