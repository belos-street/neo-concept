# 07 — 用户故事：模块 E/F — 学习统计 & 数据管理

> 所属：Neo Concept 结构化需求文档

---

## US-E1 — 查看学习数据

> **作为** 学习者
> **我希望** 查看我的学习统计数据
> **以便** 了解自己的学习进度和薄弱环节

### 验收标准

- [ ] AC-E1.1：显示总学习时长（小时/分钟）
- [ ] AC-E1.2：显示完成课时数 / 总课时数
- [ ] AC-E1.3：各模块（填词/词汇/听力/阅读/口语）正确率趋势
- [ ] AC-E1.4：词汇掌握量（已学习词汇数）
- [ ] AC-E1.5：最近学习记录（最近 10 条）

---

## US-F1 — 导出学习数据

> **作为** 学习者
> **我希望** 导出我的学习进度和数据
> **以便** 备份或迁移到新设备

### 验收标准

- [ ] AC-F1.1：导出内容：所有进度数据 + 统计数据 + 已下载课程 JSON
- [ ] AC-F1.2：导出格式：JSON（数据）或 ZIP（含课程 JSON + 图片）
- [ ] AC-F1.3：导出文件保存到设备本地存储，用户可分享/传输

### 异常流程

- **存储空间不足**：提示「存储空间不足，无法导出」
- **导出过程中 App 退出**：文件不完整，下次导出时重新生成

---

## US-F2 — 导入学习数据

> **作为** 学习者
> **我希望** 从导出文件恢复我的学习数据
> **以便** 换设备后继续学习

### 验收标准

- [ ] AC-F2.1：选择导出文件 → 校验格式 → 导入进度 + 统计 + 课程内容
- [ ] AC-F2.2：导入前显示预览（将导入 X 课进度、Y 课课程），用户确认后执行
- [ ] AC-F2.3：导入后覆盖当前本地数据

### 异常流程

- **文件格式不正确**：提示「无效的备份文件」
- **文件版本不兼容**：提示版本信息，是否强制导入（可能丢失部分数据）
