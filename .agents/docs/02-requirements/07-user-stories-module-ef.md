# 07 — 用户故事：模块 E/F/G — 学习统计 & 数据管理 & 设置

> 所属：Neo Concept 结构化需求文档

---

## US-E1 — 查看学习数据

> **作为** 学习者
> **我希望** 查看我的学习统计数据
> **以便** 了解自己的学习进度

### 验收标准

- [ ] AC-E1.1：显示总学习时长（小时/分钟）
- [ ] AC-E1.2：显示完成课时数 / 总课时数
- [ ] AC-E1.3：最近学习记录（最近 10 条，含日期、课程名、Step 进度）

### 空状态

- 无任何学习数据时，显示引导提示「开始你的第一课吧！」

---

## US-F1 — 导出学习数据

> **作为** 学习者
> **我希望** 导出我的学习进度和数据
> **以便** 备份或迁移到新设备

### 验收标准

- [ ] AC-F1.1：导出内容：所有进度数据 + 统计数据 + 已下载课程 JSON
- [ ] AC-F1.2：导出格式：JSON（数据）或 ZIP（含课程 JSON + 图片）
- [ ] AC-F1.3：导出文件保存到设备本地存储，用户可分享/传输

### 异常流程

- **存储空间不足**：提示「存储空间不足，无法导出」
- **导出过程中 App 退出**：文件不完整，下次导出时重新生成

---

## US-F2 — 导入学习数据

> **作为** 学习者
> **我希望** 从导出文件恢复我的学习数据
> **以便** 换设备后继续学习

### 验收标准

- [ ] AC-F2.1：选择导出文件 → 校验格式 → 导入进度 + 统计 + 课程内容
- [ ] AC-F2.2：导入前显示预览（将导入 X 课进度、Y 课课程），用户确认后执行
- [ ] AC-F2.3：导入后覆盖当前本地数据

### 异常流程

- **文件格式不正确**：提示「无效的备份文件」
- **文件版本不兼容**：提示版本信息，是否强制导入（可能丢失部分数据）

## US-G1 — 应用设置

> **作为** 学习者
> **我希望** 在统一的位置管理 App 配置
> **以便** 按自己的偏好调整学习参数

### 设置项清单

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 学习模式 | 线性 | 线性 / 自由 |
| TTS 朗读速度 | 1.0x | 0.5x / 0.75x / 1.0x / 1.25x / 1.5x |
| 口语评分严格模式阈值 | 60% | 可调范围 40%-90% |

### 验收标准

- [ ] AC-G1.1：设置页从底部 Tab「设置」进入
- [ ] AC-G1.2：所有设置项即时生效，无需重启 App
- [ ] AC-G1.3：设置项持久化到本地存储（key: `settings`），App 重启后保留

### 异常流程

- **本地存储读写失败**：使用默认值，设置不持久化但 App 正常运行
