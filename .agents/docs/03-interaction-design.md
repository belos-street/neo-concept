## 核心页面交互

### 1. 课程列表页 (CourseListScreen)

**布局：**
```
┌─────────────────────────────┐
│ ← 课程列表        [检查更新] │
├─────────────────────────────┤
│ ▼ Book 1: Foundations        │
│   ▼ Unit 1: Simple Present   │
│     🟢 1-1 A Day at Market  │
│     ⬜ 1-2 Shopping List     │
│     🔒 1-3 At the Zoo       │
│   ▼ Unit 2: Past Tense       │
│     🔒 2-1 Yesterday...      │
│ ▼ Book 2: Intermediate       │
│   ...                        │
└─────────────────────────────┘
```

**交互规则：**

| 元素 | 状态 | 行为 |
|------|------|------|
| 课程行 | 🟢 已下载 | 点击 → 进入学习（或显示 ResumeOverlay） |
| 课程行 | ⬜ 未下载 | 点击 → 进入下载流程 |
| 课程行 | 🔒 线性锁定 | 点击 → 提示「请先完成前一课」 |
| 课程行 | 🔄 有更新 | 显示角标，点击 → 弹窗确认更新 |
| Book/Unit 行 | — | 点击展开/折叠 |
| [检查更新] 按钮 | — | 拉取新 manifest，对比差异，标记有更新的课 |

**首次启动特殊流程：**
```
App 启动 → 从内置 URL 拉取 manifest
→ 成功：显示课程列表
→ 失败：错误页 + 重试按钮
```

### 2. 下载进度页 (DownloadScreen)

**布局：**
```
┌─────────────────────────────┐
│ ← 正在下载                   │
├─────────────────────────────┤
│                             │
│    ████████░░░░  65%        │
│                             │
│    lesson-1-2.json          │
│    预计剩余: 3s              │
│                             │
│        [取消]               │
│                             │
└─────────────────────────────┘
```

**交互规则：**
- 下载完成 → 自动跳转到 Step 1
- 取消 → 返回课程列表，该课恢复 ⬜ 状态
- 失败 → 错误提示 + 重试按钮
- 多课排队时显示「排队中 2/3」

### 3. 继续学习弹窗 (ResumeOverlay)

**触发条件：** 点击已下载且有进度的课程

**布局：**
```
┌─────────────────────────────┐
│                             │
│    1-1 A Day at the Market  │
│                             │
│    上次学到: Step 3 词汇记忆  │
│    已完成: Step 1 ✓ Step 2 ✓ │
│                             │
│    [继续学习 Step 3]         │
│    [回顾已完成内容]           │
│                             │
└─────────────────────────────┘
```

**交互规则：**
- [继续学习] → 跳转到 currentStep
- [回顾已完成] → 进入 Step 1（只读模式）
- 全部完成的课程不显示此弹窗，直接进入 Step 1（只读）

## Step-by-Step 学习界面

### 顶部 Step 进度条（所有 Step 共用）

```
┌─────────────────────────────┐
│ ← 1-1 A Day at the Market   │
├─────────────────────────────┤
│ ①──②──③──④──⑤──⑥           │
│ 预习 填词 词汇 听力 阅读 口语  │
└─────────────────────────────┘
```

- 已完成 Step 显示 ✅，当前 Step 高亮，未完成 Step 灰色
- Step 间不可跳转（只读回顾除外）

### Step 1 — 课文预习 (Step1Passage)

**布局：**
```
┌─────────────────────────────┐
│ ← Step 1 课文预习   1/6      │
├─────────────────────────────┤
│ [AI 配图区域 - 可选]          │
├─────────────────────────────┤
│ A Day at the Market          │
│                             │
│ ▶ PLAY SENTENCES             │
│                             │
│ Every Sunday, Anna goes to   │
│ the market. She buys fresh   │
│ vegetables and fruit...      │
│                             │
│ ┌─ NEW WORDS ──────────────┐ │
│ │ market   /ˈmɑːrkɪt/  市场│ │
│ │ fresh    /freʃ/     新鲜的│ │
│ └──────────────────────────┘ │
│                             │
│ ┌─ 语法点 ─────────────────┐ │
│ │ Simple Present Tense     │ │
│ │ 表示习惯性动作...         │ │
│ └──────────────────────────┘ │
├─────────────────────────────┤
│         [下一步 →]           │
└─────────────────────────────┘
```

**交互：**
- 点击任意单词 → 弹出 WordTooltip（ECDICT 释义弹窗）
- [PLAY SENTENCES] → 逐句朗读，按钮变为 ⏸ PAUSE，当前朗读句高亮
- 暂停后按钮变为 ▶ CONTINUE，点击继续播放
- 朗读中显示 ■ STOP 按钮，点击停止回到初始状态
- 词汇卡片展示本课 newVocabulary（词 + 音标 + 释义）
- TTS 速度在全局设置页面配置（0.75× / 1.0× / 1.25× / 1.5×）
- [下一步] → 标记 Step 1 completed，进入 Step 2

### Step 2 — 填词练习 (Step2FillBlanks)

**布局：**
```
┌─────────────────────────────┐
│ ← Step 2 填词练习   2/6      │
├─────────────────────────────┤
│ 1/6                          │
│                             │
│ Every ___, Anna goes to      │
│ the ___.                     │
│                             │
│ 提示: S___ (n.) 星期日       │
│                             │
│ ┌─────────────────────────┐ │
│ │ S u n d a y             │ │
│ └─────────────────────────┘ │
│                             │
│ [提交]                       │
├─────────────────────────────┤
│ 正确率: 3/6  50%             │
└─────────────────────────────┘
```

**交互：**
- 输入后实时比对（不需点提交也可即时反馈，但保留提交按钮用于确认）
- 正确 → 绿色 ✅ + 自动跳到下一空
- 错误 → 红色显示正确答案 + 🔊 发音按钮
- 全部完成 → 显示总分 + 错误词列表
- ≥ 60% → [下一步] 按钮可点击
- < 60% → [重做] 按钮（最多 3 次），3 次后仍 < 60% → 标记 completed，[下一步] 可点击

### Step 3 — 词汇记忆 (Step3Vocabulary)

#### 闪卡子模式 (FlashcardMode)

**布局：**
```
┌─────────────────────────────┐
│ ← Step 3 词汇记忆   3/6      │
│ 闪卡 (3/20)        [跳过认识]│
├─────────────────────────────┤
│                             │
│     ┌───────────────────┐   │
│     │                   │   │
│     │    market         │   │
│     │    /ˈmɑːrkɪt/     │   │
│     │    🔊             │   │
│     │                   │   │
│     │  ← 不认识  认识 →  │   │
│     └───────────────────┘   │
│                             │
│     ◀ 15/20 ▶               │
└─────────────────────────────┘
```

**交互：**
- 左滑 = 不认识（卡片飞出左侧），右滑 = 认识（卡片飞出右侧）
- 点击卡片 → 翻转显示中文 + 例句
- [跳过认识词] 开启后，已标记「认识」的词自动跳过
- 所有卡片滑完 → 进入拼写子模式

#### 拼写子模式 (SpellingMode)

**布局：**
```
┌─────────────────────────────┐
│ ← Step 3 拼写 (5/20)         │
├─────────────────────────────┤
│                             │
│         🔊 播放发音          │
│                             │
│ ┌─────────────────────────┐ │
│ │ m _ _ k _ t             │ │
│ └─────────────────────────┘ │
│                             │
│ 提示: n. 市场               │
│                             │
│        [提交]               │
└─────────────────────────────┘
```

**交互：**
- TTS 自动播放当前词发音
- 用户输入拼写 → [提交] → 即时反馈
- 正确 → 绿色 ✅，自动下一词
- 错误 → 红色高亮差异字符，显示正确拼写
- 可随时点 🔊 重听
- 全部完成 → 正确率 ≥ 60% 通过；< 60% 可重做（最多 3 次）

#### 匹配子模式 (MatchingMode)

**布局：**
```
┌─────────────────────────────┐
│ ← Step 3 匹配   3/6          │
│ 用时: 00:23                   │
├─────────────────────────────┤
│                             │
│  market    ┌──────────┐     │
│  fresh     │  市场     │     │
│  vegetable │  新鲜的   │     │
│  Sunday    │  蔬菜     │     │
│            │  星期日   │     │
│            └──────────┘     │
│                             │
└─────────────────────────────┘
```

**交互：**
- 左侧英文列，右侧中文列（随机打乱）
- 点击英文 → 再点击中文 → 配对
- 正确 → 两项变灰 + 连线动画
- 错误 → 抖动 + 红色闪烁
- 全部配对完成 → 显示用时 → 进入 Step 4

### Step 4 — 听力练习 (Step4Listening)

**布局：**
```
┌─────────────────────────────┐
│ ← Step 4 听力练习   4/6      │
│ 1/5                           │
├─────────────────────────────┤
│                             │
│    🔊 [播放音频片段]          │
│                             │
│ Q: Where does Anna go        │
│    every Sunday?             │
│                             │
│  ○ A. School                 │
│  ○ B. Market                 │
│  ○ C. Park                   │
│  ○ D. Library                │
│                             │
│        [下一题 →]             │
└─────────────────────────────┘
```

**交互：**
- 进入自动 TTS 播放当前题 audio_segment
- 可点 🔊 重复播放（不限次数）
- 选择选项 → 即时判分（正确绿色，错误红色 + 显示正确答案）
- [下一题] → 停止当前音频，切到下一题
- 播放中点下一题 → 停止当前音频
- 全部完成 → 正确率 ≥ 60% 通过；< 60% 可重做（最多 3 次）

### Step 5 — 阅读理解 (Step5Reading)

**布局：**
```
┌─────────────────────────────┐
│ ← Step 5 阅读理解   5/6      │
├─────────────────────────────┤
│ [课文全文 - 可滚动]           │
│ Every Sunday, Anna goes...   │
├─────────────────────────────┤
│ 1/5                           │
│ Q: What does Anna buy?       │
│  ○ A. Meat and rice          │
│  ● B. Vegetables and fruit   │ ← 选错后高亮
│  ○ C. Bread and milk         │
│  ○ D. Fish and chips         │
│                             │
│ ✗ 正确答案: B                 │
│ 原文: "She buys fresh        │
│  vegetables and fruit"       │ ← evidence 高亮
│                             │
│        [下一题 →]             │
└─────────────────────────────┘
```

**交互：**
- 上半区课文全文可滚动，下半区题目
- 选错 → 原文对应句高亮（evidence）+ 显示解析
- 选错的选项标红，正确选项标绿
- 全部完成 → 正确率 ≥ 60% 通过；< 60% 可重做（最多 3 次）

### Step 6 — 口语练习 (Step6Speaking)

**布局：**
```
┌─────────────────────────────┐
│ ← Step 6 口语练习   6/6      │
│ 2/5  [简单] [中等] [困难]     │
├─────────────────────────────┤
│                             │
│ "She buys fresh vegetables   │
│  and fruit."                 │
│                             │
│         🔊 播放标准发音       │
│                             │
│    ┌─────────────────┐      │
│    │    🎙️ 按住录音    │      │
│    └─────────────────┘      │
│                             │
│ 识别结果: "She buy fresh     │
│  vegetable and fruits"       │
│                             │
│ 准确率: 68%  ████████░░      │
│ 偏差词: buy→buys vegetable   │
│ →vegetables fruits→fruit    │
│                             │
│  ✅ 达标  [重录] [下一句 →]   │
└─────────────────────────────┘
```

**交互：**
- 进入自动 TTS 播放标准发音
- 按住录音按钮 → 录音（松手结束，或 30s 自动停止）

- 识别为空 → 提示「未检测到语音，请重试」
- 全部完成 → 标记课程已完成 🎉

## 全局组件交互

### WordTooltip（单词释义弹窗）

**触发：** 点击任意英文文本中的单词

**布局：**
```
┌─────────────────────────────┐
│ market  /ˈmɑːrkɪt/  🔊      │
│ n. 市场                      │
│ "She goes to the market      │
│  every Sunday."              │
└─────────────────────────────┘
```

**交互规则：**
- 点击单词 → tooltip 出现在单词下方
- 点击外部区域或再次点击该单词 → 关闭
- 点击 🔊 → Piper TTS 播放该词发音
- 点击标点符号 → 不触发
- ECDICT 未加载 → 显示「词典加载中...」
- 词不在 ECDICT → 显示「未找到该词」

### TTS 播放控制条

**复用场景：** Step 1 全文/逐句、Step 4 听力、Step 6 口语标准发音

```
[🔊 播放] [⏸ 暂停] [0.75x ▼]
```

- 同一时间只允许一个 TTS 实例播放
- 切换播放源 → 自动停止上一个
- App 切后台 → 停止播放

**布局：**
```
┌─────────────────────────────┐
│ 设置                        │
├─────────────────────────────┤
│                             │
│  学习模式                    │
│  ┌─────────────────────────┐│
│  │  线性模式         ● ○   ││
│  │  自由模式               ││
│  └─────────────────────────┘│
│                             │
│  TTS 朗读速度               │
│  ┌─────────────────────────┐│
│  │  ◀ 0.75x  1.0x  1.5x ▶ ││
│  └─────────────────────────┘│
│                             │
│  口语评分阈值                │
│  ┌─────────────────────────┐│
│  │  ████████░░  60%        ││
│  │  40% ←────────────→ 90% ││
│  └─────────────────────────┘│
│                             │
│  数据管理                    │
│  ┌─────────────────────────┐│
│  │  导出学习数据 →          ││
│  ├─────────────────────────┤│
│  │  导入学习数据 →          ││
│  └─────────────────────────┘│
│                             │
│  关于                       │
│  ┌─────────────────────────┐│
│  │  版本    1.0.0          ││
│  │  开源协议  GPL-3.0      ││
│  └─────────────────────────┘│
│                             │
└─────────────────────────────┘
```

**交互规则：**

| 设置项 | 控件 | 行为 |
|--------|------|------|
| 学习模式 | Segmented control（线性/自由） | 即时生效，切换回线性时重新锁定未完成前置课 |
| TTS 朗读速度 | 水平选项组（0.5x / 0.75x / 1.0x / 1.25x / 1.5x） | 即时生效，选中项高亮 |
| 口语评分阈值 | Slider（40%-90%，步长 5%） | 即时生效，数值显示在右侧 |
| 导出学习数据 | 点击行 → 系统文件选择器 | 导出 JSON/ZIP，成功后 Toast 提示 |
| 导入学习数据 | 点击行 → 系统文件选择器 → 预览弹窗 → 确认 | 导入前显示「将导入 X 课进度、Y 课课程」，确认后覆盖 |
| 版本/开源协议 | 纯展示 | 不可点击 |

**异常流程：**
- 导出存储空间不足 → 提示「存储空间不足，无法导出」
- 导入文件格式不正确 → 提示「无效的备份文件」
- 导入文件版本不兼容 → 提示版本信息，确认后强制导入

## 页面跳转与返回逻辑

| 场景 | 行为 |
|------|------|
| Step 内按系统返回 | 弹出确认「确定退出？当前 Step 进度不会保存」 |
| Step 完成后按返回 | 回到课程列表 |
| 课程列表按系统返回 | 退出 App（Android 双击返回） |
| 下载中按返回 | 弹出确认「取消下载？」 |
| 全文完成后进入课程 | 直接显示所有 Step（只读模式） |
