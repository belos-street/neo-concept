# 精细化任务拆解 & 单人项目管理

## 核心能力
- WBS工作拆解
- 模块拆分
- 任务颗粒度控制
- 优先级排序
- 迭代规划
- 看板化管理

## Vibe结合
每次只执行一个小任务，目标明确，持续进入编码心流。

## 核心产出
- 结构化TodoList
- 迭代计划表
- 任务看板

## 使用指南

### 1. WBS工作拆解
- 按模块拆分
- 按功能拆分
- 按接口拆分
- 按页面拆分

### 2. 任务颗粒度
- 单个任务不超过4小时
- 可独立测试
- 可独立提交
- 目标明确

### 3. 优先级排序
- P0：核心功能
- P1：重要功能
- P2：次要功能
- P3：优化功能

### 4. 迭代规划
- Sprint周期（1-2周）
- 迭代目标
- 任务排期（单人容量）
- 验收标准

### 5. 看板管理
- 待办
- 进行中
- 已完成
- 阻塞中

## AI提问模板
```
基于以下PRD和API文档，请帮我：
1. 拆解为可开发的最小任务单元
2. 按优先级排序
3. 规划迭代计划
4. 生成任务看板
```
