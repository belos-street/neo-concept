# 业务交互原型设计

## 核心能力
- 业务流程图
- 用户旅程图
- 低保真原型
- 页面跳转逻辑
- 功能闭环设计
- 交互规则定义

## 工具
- Figma/墨刀（独立开发者轻量化）

## Vibe结合
交互定死，后续UI、架构、代码全部有据可依。

## 核心产出
- 完整可交互低保真原型
- 流程图表
- 交互规则文档

## 使用指南

### 1. 业务流程图
- 用户入口
- 核心操作路径
- 分支流程
- 异常处理路径

### 2. 用户旅程图
- 用户目标
- 操作步骤
- 情绪曲线
- 痛点识别

### 3. 低保真原型
- 页面布局
- 元素位置
- 交互热区
- 状态变化

### 4. 页面跳转逻辑
- 路由规划
- 参数传递
- 返回逻辑
- 深链接支持

### 5. 交互规则
- 按钮状态
- 加载状态
- 错误提示
- 成功反馈
