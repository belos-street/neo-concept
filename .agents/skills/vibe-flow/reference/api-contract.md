# API契约优先设计

## 核心能力
- RESTful/接口规范
- 全局统一入参出参
- 错误码体系
- 鉴权契约
- 接口路由规划
- 参数校验规则

## Vibe结合
先定接口再写代码，前后端同步开发，AI可直接按契约生成接口代码。

## 核心产出
- 完整API文档
- 全局接口约束规范
- 错误码体系
- 鉴权方案
